hook

ndoojs支持在action执行之前和执行之后执行指定的操作

1. 前置操作

前置操作由actionName+'Before'组成。例如：

indexBefore: function(params){
  alert('index before');
},
 
indexAction: function(params){
  var $content;
  $content = $('#content');
  $content.html('module: home, action: indexAction);
}

2. 后置操作

后置操作由actionName+'After'组成。例如：

indexAfter: function(params){
  alert('index after');
},
 
indexAction: function(params){
  var $content;
  $content = $('#content');
  $content.html('module: home, action: indexAction);
}
