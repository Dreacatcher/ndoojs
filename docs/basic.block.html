块级组件

ndoojs支持块级组件，可以方便的定义一些按需载入自执行组件。

1. 引入do.js以及模块定义文件

<script src="../lib/Do.js"></script>

2. 添加模块定义

此处用到了事件系统，参见event。

(function(_n){
  Do.define('home', {
    path: 'example2.js',
    type: 'js'
  });
  _n.on('NAPP_DEFINE', function(){
    _n.setApp('home');
  });
})(this.N = this.ndoo || (this.ndoo = {}));
